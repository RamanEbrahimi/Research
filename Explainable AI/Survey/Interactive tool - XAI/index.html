Interactive Linear Regression Tool

Algorithm Explanation

This tool uses a linear regression model with a threshold classifier. The weights for the features are predefined, and users can allocate a total of 10 hours to adjust the starting feature values. The prediction is based on the following formula:

  • Initial Features: [60, 40, 60, 65]
  • Weights: [0.5, 0.3, 0.2, 0.4]
  • Threshold: 50 (Approved/Rejected)
  • Cost Function: The score increments depend on the hours allocated.
